Records the visual at the chosen resolution/orientation. Settings, play, record, progress and countdown are never captured. Output is an MP4 (H.264) or WebM file. Recording plays the clip from the start and stops when it ends; use the pause button to pause/resume without adding paused time to the video.

Editor tip: use 30 fps for the most reliable import into DaVinci Resolve and similar editors. Browser recordings are variable-frame-rate; if an import still misbehaves, re-encode to constant frame rate, e.g. ffmpeg -i in.mp4 -c:v libx264 -r 30 -pix_fmt yuv420p -c:a aac -b:a 192k out.mp4.

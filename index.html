⚡ Fast export (faster than real-time)
Mute audio while recording

Records the visual at the chosen resolution/orientation. Settings, play, record, progress and countdown are never captured. Output is an MP4 (H.264) or WebM file. Recording plays the clip from the start and stops when it ends; use the pause button to pause/resume without adding paused time to the video. Mute audio while recording stops it playing through your speakers (the file still gets the audio) — handy for video backgrounds; note real-time recording only renders in the foreground, so for hands-off/background use prefer Fast export. Only your loaded clip is ever recorded — other tabs (e.g. Spotify) are never captured.

Fast export renders the whole clip offline as quickly as your machine can encode (no need to wait the full duration) and the file still plays at normal 1× length. It is silent (nothing plays through your speakers) and keeps running in a background tab — start it, then switch tabs or listen to Spotify freely. Works with still backgrounds (image / preset) and needs a Chromium browser; if a background/overlay video is in use it falls back to normal real-time recording. Tip: for hands-off background exports use the Browser save mode (the Ask/Folder dialog can't open while the tab is hidden).

Editor tip: use 30 fps for the most reliable import into DaVinci Resolve and similar editors. Browser recordings are variable-frame-rate; if an import still misbehaves, re-encode to constant frame rate, e.g. ffmpeg -i in.mp4 -c:v libx264 -r 30 -pix_fmt yuv420p -c:a aac -b:a 192k out.mp4.

Batch queue
⚡ Fast export (offline)
none

Queue up to 10 audio files. Bulk-adding images pairs them in order (item 1 ↔ image 1); click any row to set that item's background individually — Current / Image / Preset, plus its own blur and zoom. Batch export renders each as its own video named <audio>-vid, saved to the chosen folder (or downloaded).

⚡ Fast export (on) renders offline — much faster than real time, silent, and keeps going in a background tab, but it can't include a background/overlay video. Turn it off for Normal export, which records in real time (full duration, tab must stay in front) and does support video layers. While a batch runs, use the pause button to pause/resume and the record button to stop.

Presets (looks)

Presets save the full look (colors, shape, background, positions, resolution, etc.) on this browser. Export/Import moves settings between browsers or backs them up. The audio clip is never included.

Background & profile images and all appearance settings are remembered on this browser. The audio clip must be re-added each session.
